# Version History

## [1.6.1] - 2026-01
### Added
- Hover on buttons

### Fixed
- Removed static css colors and replaced with variables
- Fixed linear gradients

## [1.6.0] - 2026-01
### Added
- Drop-down menu with Settings and About dialogs
- Color theme customization via Settings
- App info via About

### Improved
- Export and Import buttons moved to drop-down menu
- Less text in footer, moved to About

---

## [1.5.0] – 2025-09
### Added
- Version number synced from meta tag to UI
- Auto-resizing note fields (textarea)

### Improved
- Layout optimizations

### Fixed
- Tabs wrap to multiple rows instead of overflowing
- Textarea height resetting when switching tabs
- Background disappearing when scrolling long tabs
- PWA caching inconsistencies

## [1.4.0] – 2025-09
### Added
- Drag & drop reordering of fields
- Drag & drop tab reordering
- Search/filter inside tabs
- Inline SVG icons for edit and delete buttons

### Improved
- Styling improvements
- Better mobile and narrow-window ergonomics

## [1.3.0] – 2025-08
### Added

- LocalStorage persistence

## [1.2.0] – 2025-08
